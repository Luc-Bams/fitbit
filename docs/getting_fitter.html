1 Fitbit Time-Series Data

*describe how the data were collected, stored, and what they look like*
All the data that I’m using can be retrieved by calling the fitbit API through the getActivitiesResourceByDatePeriod, getHeartByDateIntraday, and getWeightByDate methods. The functions I wrote to perform these GET requests can be found in this script. *How I structured the data and why. –> leads to a reference to the script I used.* The code to collect new data from the fitbit servers and tidy it can be found in this script.

  • Minute-level
    • Continuous
      • calories
      • distance
      • elevation
    • Integer
      • floors
      • heartrate
      • steps
    • Categorical
      • activity intensity
  • Daily-level
    • Continuous
      • weight
      • BMI
    • Integer
      • activity calories
      • base metabolic heartrate calories
      • rest heartrate
      • minutes (per heartrate zone –> separate table) [backlog]
      • calories out (per heartrate zone –> separate table) [backlog]

1.1.1 Descriptive Statistics

min_hr <- fitbit_data$ml_int %>%
  filter(type == "heartrate") %>%
  select(value) %>%
  min()

max_hr <- fitbit_data$ml_int %>%
  filter(type == "heartrate") %>%
  select(value) %>%
  max()

fitbit_data$ml_int %>%
  filter(type == "heartrate") %>%
  ggplot(aes(x = value, fill = value, stat(density))) +
  geom_histogram(binwidth = 1, color = "black", fill = "#2A211C") +
  geom_density(alpha = 0.8, color = "black", fill = "#9F2042") +
  theme_classic() +
  labs(x = "Heart rate", y = "Density") +
  scale_x_continuous(breaks = c(min_hr, 50, 75, 100, 125, 150, 175, max_hr))

ds_histogram_heartrate 

freq_total <- fitbit_data$ml_int %>%
  filter(type == "heartrate") %>%
  nrow()

fitbit_data$ml_int %>%
  filter(type == "heartrate") %>% 
  group_by(value) %>% 
  mutate(density = n()) %>%
  ungroup() %>%
  mutate(density = density/freq_total) %>%
  ggplot(aes(x = type, y = value, color = density)) +
  theme_minimal() +
  scale_color_viridis_c(option = "D") +
  geom_jitter(alpha = 0.03, height = 0, width = 0.6) +
  geom_violin(alpha = 0, position = position_dodge(width = .75), size = 1, color = "black") +
  geom_boxplot(notch = FALSE,  outlier.size = -1, color = "black", lwd = 1, alpha = 0, width = 0.04)
